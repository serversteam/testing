Create a new CM category

1.To create a new content module, click on the entries Mailings and Content manager in the navigation bar and then on the tab CM categories and subsequently on the link New CM Category.
2.In the input field Name, give the CM category a meaningful title. This designation appears later in the overview list of CM categories. Select as clear a name as possible to simplify your work.
3.This also applies to the Description input field. The text should consist of one or two sentences and briefly describe the function of the CM category.
4.OpenEMM accepts all of the settings with Save.
