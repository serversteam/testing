Other profile data

The More profile data section contains fields for internal use as well as all user-defined fields for the recipients’ profiles. Fields for internal use are:

•CHANGE_DATE: Date of last profile change.
•CREATION_DATE: Date of recipient profile creation.
•DATASOURCE_ID: where the data were imported from – see chapter Import function for recipient data.

User-defined fields follow. This list can be of any length and is based solely on your needs and requirement criteria, for instance customer IDs or other reference numbers.
