Auto-Import

A CSV file with recipient data that is stored on a server can be automatically imported into OpenEMM. To do so, you must set up an Auto-Import. In the navigation bar below Recipient, select the Auto Import section and switch to the New Auto Import tab. After entering a Name and Description, you can select an Import profile that has previously been saved under Recipient, Import (see Chapter Creating a new import profile). Then inform OpenEMM of the File server and the File path for the recipient data to be imported.

Make sure that you write this correctly. The file server must be input according to the following syntax:

username:password@domain

For the file path, enter the complete service directory including the filename, for example:

/home/user/file.csv

If you want to make a one-time import, place a tick next to Execute one time only. Placing a tick next to Skip faulty records causes OpenEMM to carry out the import even if there are errors in the data. In any other case the import will be terminated.

Now determine the day and time when the import is to be started as well as the Mailing lists in which you want to set the recipient data to active. Multiple selection is possible here – in theory you can execute the import every day at a specific time as well as select multiple mailing lists for the imported recipient data. Once you have made your selection, confirm it by clicking on Save.

New buttons will now appear next to the Save button. After clicking Activate, the Auto-Import is executed at the set time, while Deactivate stops the Auto-Import. Clicking on Import recipients causes OpenEMM to execute the export  once and immediately. After every import, the user given in the import profile receives a summary via e-mail.

Fig. 4.23: Auto-Import assigns recipient data automatically to one or more mailing lists at scheduled times.

In the Overview tab, OpenEMM shows you a list of all created Auto-Imports including the given description, the file server and the status, which can be set to be either active or inactive. By clicking on a list entry you can edit the corresponding profile as described above. If you no longer need an Auto-Import, you can either delete it in the Overview tab by clicking on the wastebasket symbol or also delete it in the detailed settings by means of the Delete button.
