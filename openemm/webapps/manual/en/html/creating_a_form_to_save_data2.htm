Creating a form to save data

This section can be very short since both the form and the action are direct equivalents of saving data after amending a customer record. Chapters Creating a new action and Creating a form to save data contain further information on how to create the form and the corresponding action. If you have already created an action for saving amendments to a customer profile, that action can also be used for saving data for a new subscriber.
