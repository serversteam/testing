Managing users

To manage existing users, in the main menu click on Administration then on User. OpenEMM now shows you a list of all existing users.

Fig. 10.8: An overview displaying all users for all clients.

To edit the settings for a created user, either click on the user name or on the Edit symbol on the right-hand end of the corresponding line. Use the Save button in the input mask to save your changes to the user settings. If you want to delete a user, either click on Delete in the input mask or click on the Wastebasket symbol in the overview.
