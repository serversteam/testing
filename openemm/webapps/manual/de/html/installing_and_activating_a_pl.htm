Plugin installieren und aktivieren

Um ein Plugin zu installieren, rufen Sie im Hauptmenü unter Verwaltung den Plugin Manager auf und klicken Sie dann auf das Register Plugin installieren.

Abb. 10.11: Ein neues Plugin integrieren Sie über diesen Dialog.

Teilen Sie dem OpenEMM den Speicherort des Plugins mit, indem Sie auf die Schaltfläche Durchsuchen klicken. Haben Sie das gewünschte Plugin ausgewählt, wird es mit Import direkt in den OpenEMM integriert. Das neue Plugin erscheint anschließend in der Liste sämtlicher Plugins.

Nun müssen Sie das Plugin noch aktivieren. Dazu klicken Sie in der Übersicht auf die entsprechende Zeile oder auf das ganz rechts in der Zeile stehende Icon. Im folgenden Fenster erscheinen noch einmal die wichtigsten Detailinformationen zum Plugin sowie die beiden Schaltflächen aktivieren und deinstallieren. Klicken Sie auf erstere Schaltfläche, um das Plugin zu aktivieren. Der OpenEMM übernimmt diese Einstellung und bestätigt das mit der Rückmeldung Einstellungen gespeichert.

Abb. 10.12: Das neue Plugin startet, sobald Sie auf die Schaltfläche aktivieren klicken.
