Auto-Import

Eine CSV-Datei mit Empfängerdaten, die auf einem Server gespeichert ist, lässt sich automatisch in den OpenEMM importieren. Dazu müssen Sie einen Auto-Import erstellen. Wählen Sie in der Navigationsleiste unter Empfänger die Rubrik Auto-Import aus und wechseln Sie zum Reiter Neuer Auto-Import. Nach Vergabe von Name und Beschreibung können Sie ein Importprofil auswählen, das zuvor unter Empfänger, Import gespeichert wurde (siehe Kapitel Neues Importprofil erstellen). Anschließend teilen Sie dem OpenEMM den Dateiserver und den Dateipfad für die zu importierenden Empfängerdaten mit.

Achten Sie dabei auf die korrekte Schreibweise. Der Dateiserver muss nach folgender Syntax eingegeben werden:

benutzername:passwort@domain

Für den Dateipfad geben Sie das komplette Server-Verzeichnis samt Dateiname an, beispielsweise:

/home/user/datei.csv

Wollen Sie den Import ein einziges Mal durchführen, setzen Sie ein Häkchen bei nur einmalige Ausführung. Ein Häkchen bei Fehlerhafte Daten überspringen veranlasst den OpenEMM, den Import auch dann durchzuführen, wenn Fehler in den Daten vorliegen. Andernfalls wird er abgebrochen.

Nun legen Sie noch den Tag und die Uhrzeit fest, an dem der Import starten soll, sowie die Mailing-Listen, auf welchen Sie die Empfängerdaten aktiv setzen wollen. Hier ist auch eine Mehrfachauswahl möglich – Sie können den Import theoretisch an jedem Tag zu einer bestimmten Uhrzeit durchführen lassen, sowie mehrere Mailinglisten für die importierten Empfängerdaten auswählen. Haben Sie Ihre Auswahl getroffen, übernehmen Sie die Einstellungen mit einem Klick auf Speichern.

Nun erscheinen neben dem Speichern-Button neue Schaltflächen. Nach einem Klick auf Aktivieren wird der Auto-Import zum eingestellten Zeitpunkt ausgeführt, während Deaktivieren den Auto-Import wieder stoppt. Mit einem Klick auf Empfänger importieren führt der OpenEMM den Export sofort einmalig durch. Nach jedem Import erhält der im Importprofil angegebene User eine Zusammenfassung per E-Mail.

Abb. 5.24: Der Auto-Import weist Empfängerdaten automatisch und zeitgesteuert einer oder mehreren Mailingslisten zu.

Im Reiter Übersicht zeigt Ihnen der OpenEMM eine Liste aller angelegten Auto-Importe an, inklusive der vergebenen Beschreibung, dem Dateiserver sowie dem Status, der entweder auf aktiv oder nicht aktiv gesetzt sein kann. Mit einem Klick auf einen Listeneintrag können Sie das entsprechende Profil wie oben beschrieben bearbeiten. Benötigen Sie einen Auto-Import nicht mehr, können Sie ihn wahlweise im Reiter Übersicht durch Anklicken des Papierkorbsymbols löschen, oder in den Detaileinstellungen über die Schaltfläche Löschen.
