Die Online-Hilfe

In allen Eingabedialogen des OpenEMM steht Ihnen eine Online-Hilfe zur Verfügung. Über das Hilfe-Symbol, ein rot eingerahmtes Fragezeichen oben rechts auf der Webseite, rufen Sie die Online-Hilfe auf. Diese erscheint in einem separaten Fenster, so dass Sie parallel dazu im OpenEMM weiterarbeiten können.

Abb. 2.6: Die Online-Hilfe rufen Sie über das Fragezeichen-Symbol auf.

Mit der Online-Hilfe finden Sie die gewünschten Antworten sehr schnell: Der OpenEMM erkennt automatisch, in welchem Menü Sie die Online-Hilfe aufgerufen haben und schlägt direkt das dazu passende Kapitel auf.

Das Online-Handbuch können Sie optional auch komplett im PDF-Format herunterladen und so jederzeit als „Offline-Nachschlagewerk“ verwenden. Klicken Sie dazu in der Menüleiste der Online-Hilfe auf den Eintrag OpenEMM Handbuch.

Bitte beachten Sie: Das Hilfe-Symbol in der Top-Navigation der Webseite öffnet die Startseite der Online-Hilfe, während das Fragezeichen-Symbol in der rechten, oberen Ecke das passende Kapitel aufruft.
