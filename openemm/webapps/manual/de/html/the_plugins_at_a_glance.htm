Die Plugins im Überblick

Wenn Sie sich einen Überblick darüber verschaffen wollen, welche Plugins derzeit in den OpenEMM eingebunden sind, rufen Sie im Hauptmenü unter Verwaltung den Eintrag Plugin-Manager auf. Im folgenden Dialog listet der OpenEMM sämtliche Plugins mitsamt ihrem Status auf. Neben der Plugin-ID, dem Pluginname, einer kurzen Beschreibung, der Version und dem Anbieter sehen Sie auch, ob das entsprechende Plugin aktiviert wurde oder derzeit nicht aktiv ist.

Abb. 10.10: Mit dem Plugin Manager können Sie den OpenEMM um beliebige Funktionen erweitern.

Der OpenEMM zeigt in der Standardeinstellung 50 Listeneinträge an. Um diese Ansicht anzupassen, klicken Sie unter Listenlänge wahlweise auf die Zahl 20 oder 100 und anschließend auf Anzeigen.
