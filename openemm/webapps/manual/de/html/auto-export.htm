Auto-Export

In der Navigationsleiste finden Sie unter Empfänger auch die Rubrik Auto-Export. Mit dieser Funktion lassen sich zuvor definierte Exporte zeitgesteuert durchführen. Einen neuen automatischen Export legen Sie im Reiter Neuer Auto-Export an. Nach Vergabe von Name und Beschreibung können Sie ein Export Profil auswählen, das zuvor unter Empfänger, Export gespeichert wurde (siehe Kapitel Exportfunktion für Empfängerdaten).

Anschließend teilen Sie dem OpenEMM den Dateiserver und den Dateipfad der zu speichernden Empfängerdaten mit. Achten Sie dabei auf die korrekte Schreibweise. Der Dateiserver muss nach folgender Syntax eingegeben werden:

benutzername:passwort@domain

Für den Dateipfad geben Sie das komplette Server-Verzeichnis samt Dateiname an, beispielsweise:

/home/user/datei.csv

Wollen Sie den Export ein einziges Mal durchführen, setzen Sie ein Häkchen bei nur einmalige Ausführung. Nun legen Sie noch den Tag und die Uhrzeit fest, an dem der Export starten soll. Hier ist auch eine Mehrfachauswahl möglich – Sie können den Export theoretisch an jedem Tag zu einer bestimmten Uhrzeit durchführen lassen. Haben Sie Ihre Auswahl getroffen, übernehmen Sie die Einstellungen mit einem Klick auf Speichern.

Nun erscheinen neben dem Speichern-Button neue Schaltflächen. Nach einem Klick auf Aktivieren wird der Auto-Export nach den vorgegebenen Regeln ausgeführt, während Deaktivieren den Auto-Exportprozess wieder stoppt. Mit einem Klick auf Empfänger exportieren führt der OpenEMM den Export sofort einmalig durch.

Abb. 5.30: Der Auto-Export sichert die Empfängerdaten automatisch und zeitgesteuert.

Im Reiter Übersicht zeigt Ihnen der OpenEMM eine Liste aller angelegten Auto-Exporte an, inklusive der vergebenen Beschreibung, dem Dateiserver sowie dem Status, der entweder auf aktiv oder nicht aktiv gesetzt sein kann. Mit einem Klick auf einen Listeneintrag können Sie das entsprechende Profil, wie oben beschrieben, bearbeiten. Benötigen Sie einen Auto-Export nicht mehr, können Sie ihn wahlweise im Reiter Übersicht durch Anklicken des Papierkorbsymbols löschen, oder in den Detaileinstellungen über die Schaltfläche Löschen.
