Plugin deaktivieren und deinstallieren

Um ein aktives Plugin zu deaktivieren, wählen Sie es im Übersichtsmenü des Plugin Manager aus, indem Sie auf die entsprechende Zeile oder auf das ganz rechts in der Zeile stehende Icon klicken. Klicken Sie anschließend auf deaktivieren, um das Plugin abzuschalten. Der OpenEMM bestätigt das mit der Rückmeldung Einstellungen gespeichert.

Abb. 10.13: Mit einem Klick auf deaktivieren schaltet der OpenEMM die Funktion des Plugins ab.

Um ein Plugin aus dem OpenEMM zu entfernen, gehen Sie genauso vor wie im obigen Absatz beschrieben, klicken jedoch auf deinstallieren statt auf deaktivieren. Der OpenEMM löscht das Plugin dann aus dem System.

Bitte beachten Sie: Sie können ein Plugin nur dann deinstallieren, wenn es nicht aktiv ist.
