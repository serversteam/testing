Empfängerprofil erweitern

Im Beispiel soll das Interesse der Empfänger für zwei verschiedene Themen (Reisen und Autos) gemessen werden. Dafür sind entsprechend zwei zusätzliche Felder im Empfängerprofil erforderlich. Bei jedem Klick des Empfängers auf einen themenverwandten Link soll der Feldinhalt um 1 erhöht werden. Nach einigen Mailings ergibt sich damit ein recht klares Bild der Empfängerinteressen. Wer einmal auf einen Reiselink klickt, ist nicht so interessiert wie ein Empfänger mit 15 Klicks auf dem Konto. Entsprechend müssen Sie zwei Numerisch-Felder anlegen, die mit dem Standardwert 0 belegt sind. Als aussagekräftige Namen verwenden Sie interest_travel und interest_cars.

1.Klicken Sie in der Navigationsleiste auf die Schaltfläche Verwaltung und den Unterpunkt Felder verwalten. Der Inhaltsbereich zeigt in einer Liste alle bereits vorhandenen selbst definierten Felder. Klicken Sie auf das Register Neues Feld.
2.In das erste Eingabefeld hinter Feldname tippen Sie interest_cars ein. Bei Beschreibung tragen Sie einen ausführlichere Beschreibung ein – was im konkreten Fall nicht viel ist. Bei Feldname in DB tragen Sie interest_cars ein. Groß- und Kleinschreibung spielt keine Rolle, der OpenEMM wandelt sowieso alle Zeichen in Großbuchstaben um. In der Auswahlliste Typ stellen Sie Numerisch ein. Das Feld Länge kann unverändert bleiben, das System stellt immer 22 Zeichen ein. Als Standard-Wert tragen Sie im letzten Eingabefeld eine 0 ein. Damit haben alle Empfängerprofile denselben Startwert.
3.Schließen Sie die Eingabe mit einem Klick auf die Schaltfläche Speichern ab.

Abb. 8.35: Legen Sie ein neues Numerisch-Feld Interesse_Autos an.

4.Verfahren Sie genauso mit dem zweiten Feld für Reisen. Tragen Sie als Feldname interest_travel ein, als Feldname in DB interest_travel, als Typ Numerisch und als Standard-Wert eine 0. Schließen Sie auch hier mit der Schaltfläche Speichern ab.
