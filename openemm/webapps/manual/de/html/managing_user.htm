Benutzer verwalten

Um die bestehenden Benutzer zu verwalten, klicken Sie im Hauptmenü unter Verwaltung auf Benutzer. Der OpenEMM zeigt Ihnen nun eine Liste sämtlicher angelegter Nutzer an.

Abb. 10.8: Alle Benutzer ind in einer Übersicht zusammengefasst.

Um die Einstellungen für einen angelegten Benutzer zu bearbeiten, klicken Sie entweder auf dessen Benutzer-Namen oder auf das Bearbeiten-Symbol am rechten Ende der entsprechenden Zeile. Ihre Änderungen an den Benutzereinstellungen übernehmen Sie in der Eingabemaske mit Speichern. Wollen Sie einen Benutzer löschen, klicken Sie entweder in der Eingabemaske auf Löschen oder in der Übersicht auf das Papierkorb-Symbol.
