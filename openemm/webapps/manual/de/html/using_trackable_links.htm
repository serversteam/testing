Messbare Links verwenden

Für die Auswertung eines Mailings lassen sich im Text messbare Links integrieren. Klickt ein Empfänger des Mailings den Link an, ruft er damit eine URL auf, die auf den AGNITAS-Server verweist. Der Server registriert den Mausklick und ordnet ihn dem speziellen Empfänger zu. Das funktioniert sowohl bei der Text-Version als auch bei der HTML-Version. Allerdings ist die Schreibweise für Text-Mails anders als für HTML-Mails. Das Beispiel enthält bereits einen entsprechenden Link. Entsprechend gelten hier die Aussagen zur Text-Version wie in Kapitel Einen Link in der Text-Version erstellen.

Ein messbarer Link stellt fest, ob ein bestimmter Empfänger eines Mailings diesen Link angeklickt hat. So stellen Sie leicht fest, ob besonderes Interesse an einem bestimmten Thema besteht. Wie Sie die Ergebnisse auswerten lesen Sie in Kapitel Statistik - Überblick gewinnen.
